3.HTML

  • HTML规范

  • 引入资源使用相对路径，不要指定资源所带的具体协议 ( http:,https: ) ，除非这两者协议都不可用

  • 使用双引号来标识html的属性

    eg：

    <a class="a-recommend">推荐使用双引号</a>
<a class='a-recommend'>不推荐不推荐使用单引号</a>
